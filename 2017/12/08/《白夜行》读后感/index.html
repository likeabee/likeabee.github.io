《白夜行》读后感

发表于 2017-12-08 | 分类于 读后感 | 0 comments | 阅读次数:

emmmm…不知从何说起
最近上下班的地铁上除了打瞌睡、看知乎日报以外，我也开始看一看一些小说
东野圭吾先生的大名最近一直都在我的视线以及耳廓中流连…于是…
《白夜行》读罢，在手机上阅读的时候，常常因为看的入神都会忘记看自己在地铁上过了多久，这大概就是好的推理小说罢（大概也许应该是推理小说吧…）
东野圭吾先生的魅力在于他不仅仅是一个写推理很牛逼的逻辑严密的“理科优等生”，也因为他对环境及人性的描写和刻画甚至超过写出华丽辞藻的“文科优等生”…我不知道这样描述是否恰当…在不经意间东野圭吾先生写到当时的环境或者当时人物的动作和心理活动，回味起来都别有一番意味，令人不禁探寻是否自己在阅读时遗漏了什么
回到《白夜行》，这似乎就是一部雪穗与桐原亮司的成长过程，书中的警探屉垣润三是这样形容他们二人的关系的——枪虾与虾虎鱼，总之就是一个词语：互利共生。不过我心底里似乎不是太同意这个观点，因为在我看来，这两个人的命运从小时就被捆绑在了一起，从桐原洋介死去开始，也许更早，显然他们儿时对彼此是有好感的，但是不幸的事发生在了雪穗身上，而始作俑者居然是亮司的父亲桐原洋介…亮司弑父成了起点…
在我眼中，无论何时，从遇到（可能比较狭隘）雪穗开始直至生命结束，亮司始终深爱着雪穗。那么雪穗呢？对于雪穗而言，也许儿时的好感早就在那件事后变了味（那件事在雪穗心中芥蒂至深，导致雪穗以同样的方法伤害了多个身边的人）也许雪穗此后从来都只是在利用着亮司…按照互利共生的说法，双方必定都从对方获取利益，于雪穗而言，利益不言而喻。但至于亮司，能从雪穗处得到什么呢？钱？似乎从亮司后半部的处境来看，落魄的情况历历在目，但是明明赚了那么多钱，亮司却并没有活的如雪穗般光鲜。似乎亮司处于永远的黑夜中，大概雪穗是那轮照亮黑夜的月吧，正如书名”白夜“所言。这其实是亮司一个人的一部悲剧，当明月转身，黑夜便从此不再有光明。一份深爱。
在我读完全书后，站在书外的角度审视全篇，东野圭吾先生的功力又显现出来。雪穗与亮司的故事自然是主干，但是《白夜行》的思路是从散乱的各个角度慢慢汇入主干的，一个又一个的故事联系在一起，一颗枝叶繁茂的大树豁然眼前。不仅如此，相信看过此书的同学都有一种感觉，看到一半时，其实已经明显感到凶手其实就在我们心中了。对于一半的推理小说而言，知道凶手是谁，其实它就已经失色，但是我还是愿意甚至迫切的想要继续阅读《白夜行》，东野圭吾先生讲述手法可见一斑。给我的感受就是这本书读完其实你已经读了两遍了，虽然时间线一直在推进，但是前一半过去后，东野圭吾先生借助两位侦探，抽丝剥茧，让我们又回溯了一遍。而我并不觉得无趣，反而觉得整个故事似乎慢慢完整了…
…
为东野圭吾先生打电话…
相信我还会继续阅读先生的其他书籍…
